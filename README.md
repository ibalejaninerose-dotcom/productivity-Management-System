# 📊 Productivity Management System

## 📌 Overview
This system manages gateway loss records for telecommunication providers (Globe, Smart, DITO). It provides full CRUD operations and integrates with GitHub API for repository statistics.

## 🛠️ Technologies Used
- **Laravel 11** - Backend Framework
- **MySQL** - Database
- **Postman** - API Testing & Documentation
- **GitHub API** - External Service
- **HTML/CSS/JS** - Frontend Dashboard
- **Jira** - Project Management
